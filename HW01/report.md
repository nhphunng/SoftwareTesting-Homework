## Requirement 2 - 20 Software Defects 2022–2026

### 1. CrowdStrike global outage
- **Source link:** [CrowdStrike-related_IT_outages](https://en.wikipedia.org/wiki/2024_CrowdStrike-related_IT_outages), [Biggest IT outages of 2023–2025](https://www.xurrent.com/blog/it-outages#crowdstrike-global-outage-july-2024)
- **Description:** The global IT outage on July 19, 2024, remains the most severe computer crash in operational history. It originated within CrowdStrike’s threat intelligence system, which distributes dynamic "Channel Files" to Falcon Sensor clients running at kernel-level execution (Ring 0) on Microsoft Windows hosts. During a routine push, CrowdStrike distributed Channel File 291 containing an unvalidated configuration update.
- **Severity:** Critical because  kernel-level crashes prevent remote management tools from executing recovery actions. The consequences were catastrophic: approximately 8.5 million systems crashed simultaneously, halting operations across airlines, banks, and healthcare facilities.
- **Consequences :** Simultaneous Blue Screen of Death (BSOD) boot loops on approximately 8.5 million Windows hosts globally, triggering global logistics, aviation, financial transactions, and healthcare blackouts with financial losses exceeding $10 billion. 
- **Solution:** Implement progressive staged rollout policies, conduct rigorous sandboxed local verification testing, and establish endpoint segmentation.

### 2. AT&T Mobile Signaling Core Blackout
- **Source link:** [Biggest IT outages of 2023–2025](https://www.xurrent.com/blog/it-outages)
- **Description:** In February 2024, AT&T's nationwide cellular network suffered a massive signaling core failure. The defect occurred during a routine cellular expansion when engineers deployed an unvalidated routing parameter change to the core network registration database. This routing misconfiguration prevented mobile devices from establishing security handshakes with cellular towers, and when the network registration nodes attempted to process the initial traffic, they fell into a database synchronization deadlock, blocking mobile registrations across the continent.
- **Severity:** Critical because it blocked voice, data, and SMS services for millions of subscribers, severed public safety communication lines, and prevented critical access to 911 emergency centers.
- **Consequences:** Complete blackout of cellular services across the nation, disruption of emergency 911 routing paths, and loss of critical communication infrastructure for millions of subscribers.
- **Solution:** Implement mandatory peer reviews for cellular configurations, along with isolated routing boundaries to contain future upgrades.

### 3. Microsoft Azure Wide Area Network Configuration Escalation
- **Source link:** [Biggest IT outages of 2023–2025](https://www.xurrent.com/blog/it-outages)
- **Description:** In January 2023, Microsoft Azure experienced a severe WAN routing disruption that propagated globally. An automated utility pushed a network routing table update to a core WAN router, but due to a configuration oversight, the change was executed across all WAN nodes without regional isolation. This network update established a circular routing loop, saturating internal fibers with infinite packet replication.
- **Severity:** High because the network storm severed communication pathways between Azure's regional data centers, disrupting compute, storage, and application access across North America, Europe, and Asia.
- **Consequences:** Multi-continent disruption of Azure storage, compute, and Microsoft 365 services, causing services such as Microsoft Teams, Exchange, and SharePoint to go offline for hours and impacting millions of enterprise users.
- **Solution:** Isolate routing boundaries and deploy automated rollback scripts that trigger when packet drop rates exceed historical baselines.

### 4. FAA NOTAM Database Sync Corruption
- **Source link:** [Biggest IT outages of 2023–2025](https://www.xurrent.com/blog/it-outages)
- **Description:** The FAA was forced to order a historic ground stop of all United States domestic flights in January 2023 due to a database sync failure within the legacy NOTAM system. During routine systems maintenance, an engineer inadvertently deleted critical operational parameter files in the primary NOTAM database. Because the synchronization protocol lacked validation checks, the deletion was immediately mirrored to the secondary backup database, rendering both environments unreadable.
- **Severity:** Critical because the loss of real-time safety advisories prevented the dissemination of flight safety hazards to pilots.
- **Consequences:** Complete US domestic aviation ground stop with delay of over 10,000 flights, stranding millions of travelers and costing the aviation industry hundreds of millions of dollars.
- **Solution:** Decouple primary-secondary write actions and transition legacy databases to modern engines with validation checks.

### 5. Asana Web Runtime Telemetry Overload
- **Source link:** [The top internet outages of 2025: analyses and takeaways](https://www.thousandeyes.com/blog/the-top-internet-outages-of-2025-analyses-and-takeaways)
- **Description:** On February 5 and 6, 2025, Asana suffered two consecutive configuration-related outages. The root cause was a minor configuration change deployed to production that triggered an unhandled exception loop within core web application routers. Because the logging framework lacked asynchronous queue isolation, it attempted to write extensive stack traces to local storage on the application hosts for every failed request, exhausting disk capacity in minutes.
- **Severity:** High because it completely blocked enterprise customers from accessing project boards for several hours across consecutive business days.
- **Consequences:** Double-outage blocking corporate collaboration dashboards globally, rendering the web runtime unable to process requests.
- **Solution:** Decouple logging from runtimes and deploy asynchronous telemetry queues with strict local storage quotas.

### 6. Spotify Playback Desynchronization
- **Source link:** [The top internet outages of 2025: analyses and takeaways](https://www.thousandeyes.com/blog/the-top-internet-outages-of-2025-analyses-and-takeaways)
- **Description:** On April 16, 2025, Spotify experienced a silent backend failure where frontend indicators reported normal health, but track and video playback failed globally. The root cause was a data synchronization defect in the backend playback orchestration pipeline. A microservices update had modified the playback metadata structure, but the caching layer failed to update its indexes, causing the frontend client to request track pointers that the media-delivery servers could not resolve.
- **Severity:** Medium because although user authentication and the application UI loaded normally, the core utility of the application was completely broken.
- **Consequences:** Silent playback failure while UI indicators reported nominal health, resulting in complete loss of streaming functionality.
- **Solution:** Implement end-to-end telemetry probes verifying actual playback packet alignment rather than relying on high-level HTTP status responses.

### 7. Azure Front Door EMEA Edge Router Thread Panic
- **Source link:** [The top internet outages of 2025: analyses and takeaways](https://www.thousandeyes.com/blog/the-top-internet-outages-of-2025-analyses-and-takeaways)
- **Description:** On October 9, 2025, Microsoft Azure Front Door edge sites in the EMEA region crashed, causing widespread traffic disruptions. The incident was traced to a software parsing error in the edge router's HTTP handler. Following an edge software update, a parsing function failed to allocate sufficient memory buffers when handling highly nested HTTP request headers, triggering thread panics when such requests reached EMEA edge nodes.
- **Severity:** High because it disconnected thousands of European websites relying on Azure Front Door, routing traffic to distant regions.
- **Consequences:** Regional traffic collapse in EMEA with application latency across downstream platforms.
- **Solution:** Deploy robust input-sanitization parsing rules and redirect edge traffic automatically.

### 8. AWS DynamoDB Control Plane Propagation
- **Source link:** [The top internet outages of 2025: analyses and takeaways](https://www.thousandeyes.com/blog/the-top-internet-outages-of-2025-analyses-and-takeaways)
- **Description:** On October 20, 2025, AWS suffered a major service disruption that began in the US-EAST-1 region and quickly propagated globally. An automated routing optimization script executed a configuration change that inadvertently deleted critical DynamoDB routing tables in US-EAST-1. Several global AWS control plane systems—including IAM and DynamoDB Global Tables—contained hardcoded dependencies on the US-EAST-1 regional endpoint, causing the regional deletion to block global lookup requests.
- **Severity:** Critical because the architectural tight coupling caused the regional failure to propagate worldwide.
- **Consequences:** 15-hour outage of Slack, Atlassian, and Snapchat with approximately $581M in financial damage.
- **Solution:** Decouple global identity/table services from single regional control endpoints.

### 9. Cloudflare BGP Route Withdrawal
- **Source link:** [The top internet outages of 2025: analyses and takeaways](https://www.thousandeyes.com/blog/the-top-internet-outages-of-2025-analyses-and-takeaways)
- **Description:** On July 14, 2025, Cloudflare's 1.1.1.1 DNS and content delivery networks went offline globally. The cause was a latent configuration defect introduced during a routine network modernization update. A deferred deletion script lacked validation constraints, and when triggered, it deleted active routing parameters, causing Cloudflare's edge routers to withdraw their BGP route announcements from the global internet routing tables.
- **Severity:** High because Cloudflare's global IP addresses disappeared from the routing tables, taking thousands of downstream websites offline.
- **Consequences:** Thousands of downstream sites disconnected globally for over an hour.
- **Solution:** Implement simulation-based route validation and BGP peer checks.

### 10. Shopify Database Write-Ahead Log Deadlock
- **Source link:** [Software Bugs in 2025](https://www.testdevlab.com/blog/software-bugs-2025)
- **Description:** During the high-volume traffic window of Cyber Monday in November 2025, Shopify's backend platform experienced a major administrative system outage.
